1. Librerías utilizadas
requests
googletrans
¿Para qué sirven las librerías?
requests: permite hacer peticiones a APIs en internet para obtener información.
googletrans: permite traducir texto de un idioma a otro (en este caso de inglés a español).
Instalación

2. ¿Cómo ejecutar el programa?
(Las librerias deben ser instaladas primero)
Abrir la terminal en la carpeta del proyecto
Ejecutar el archivo principal:

python main.py

Descripción del programa

Este programa permite al usuario obtener ideas de actividades cuando está aburrido utilizando una API externa. Además, traduce automáticamente las actividades del inglés al español.

Funcionalidades
Mostrar una actividad aleatoria
Mostrar una actividad filtrada (por ejemplo, no apta para niños)
Traducir automáticamente el texto de la actividad
